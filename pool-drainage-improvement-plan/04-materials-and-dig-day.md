### 4. Dig-day sequence (order of operations)

**Before Day 1**
- [ ] Call **811 (Sunshine 811)** 2–3 business days ahead for utility locates.
- [ ] Run the **hose grade test**; confirm street vs. dry well.
- [ ] Confirm county/city right-of-way discharge rules (corner lot).
- [ ] **Mark the full layout** with paint: fence line (A), deck edge (B), pipe route (E), sump + outlet.

**Day 1 — Outlet & spine.** Dig the main conveyance trench to the outlet, **setting slope with a line level** (solid pipe ¼"/ft ideal, 1% minimum). Install the **sump catch basin** at the deck low point and the outlet (pop-up or dry well). This is the backbone everything ties into — build it first so every later line drains to it.

**Day 2 — Fence interceptor (①).** Trench along the fence, lay fabric → rock → perforated pipe (holes down) → rock → burrito-wrap fabric. Slope toward the main. Add the high-end cleanout.

**Day 3 — Deck-edge drain (②a) + lawn French drain (②b) + downspouts (④).** Set the channel drain in concrete flush with the pavers; run the lawn-side French drain; tie downspouts into the main with solid pipe. Let concrete cure.

**Day 4 — Regrade (③).** Shape the lawn so it falls **away from the deck** toward the interceptor/moat and outlet. Compact lightly. Verify fall with the level — no low spots that pond.

**Day 5 — Stabilize & test.** Lay sod; pin erosion blanket on slopes; backfill/clean all trenches; **flood-test with a hose** at the fence line and watch it travel to the outlet with nothing reaching the pool.

### 5. DIY vs. contractor — where to draw the line

| DIY-friendly | Consider a pro |
|---|---|
| Fence interceptor (①) — straightforward trench work | Setting the channel drain true in concrete (②a) |
| Downspout tie-ins (④) | Confirming outlet grade over a long/flat run (⑤) |
| Sod + erosion blanket (③, finish) | Large regrade / hauling fill (③, bulk) |
| Flood-testing | Right-of-way connection if a permit is required |

A common split: **hire out Day 1 (outlet/spine) + the channel drain**, DIY the interceptor, downspouts, regrade finish, and sod.
